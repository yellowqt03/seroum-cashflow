### 1. 할인 승인 관리 시스템 ✅

- 페이지: /approvals
- 기능: 중복 할인 요청 승인/거부

### 2. 쿠폰 관리 시스템 ✅
- API: GET/POST/PUT/DELETE /api/coupons
- 페이지: /coupons
- 기능: 쿠폰 생성, 수정, 삭제, 사용 이력 조회

### 3. 매출 리포팅 시스템 ✅
- API: /api/reports/sales, /api/reports/services
- 페이지: /reports
- 기능: 일/주/월/년 매출 분석, 서비스 판매 순위, 엑셀 업로드/다운로드

### 4. 특이사항 관리 시스템 ✅
- API: GET/POST/PUT/DELETE /api/monthly-notes
- 페이지: /notes
- 기능: 월별 특이사항 등록, 수정, 삭제, 연도 필터링

## 🚀 실행 방법

### 현재 상태
- ⚠️ Node.js 18.17.1 (18.18.0 이상 필요)
- ✅ Prisma Studio 실행 중: http://localhost:5555
- ✅ 데이터베이스 동기화 완료

### 개발 서버 실행 (Node.js 18.18+ 필요)
```bash
npm run dev
# http://localhost:3000 접속
```

### Prisma Studio (데이터베이스 관리)
```bash
npx prisma studio
# http://localhost:5555 접속
```
